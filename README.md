The main files implementing the bioelectric field model are: cellularFieldNetwork.py, simulateTrainedModel.py and simulateCellularFieldNetwork.py
To simulate the Mosaic and Stigmergic models, set the 'Model' variable in simulateTrainedModel.py to either "Stigmergic" or "Mosaic" and run the file.
To simulate arbitrary models, simply set the appropriate variables in simulateCellularFieldNetwork.py and run the file.
